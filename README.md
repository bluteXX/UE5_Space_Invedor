# 👾 Space Invaders — UE5 Project

Projekt gry typu *Space Invaders* łączący klasyczną formułę z elementami orbitalnej rozgrywki 2D (Paper2D) - statki wroga krążą po orbitach, spadają w stronę gracza, a asteroidy i wrogowie spiralni dodają chaosu. 🚀

---

## 🎮 Tryby gry

### 🛡️ Save The Earth
Broń Ziemi przed inwazją! Planeta ma swój pasek życia - każdy wróg, który przebije się przez granicę orbity, zadaje jej obrażenia. Gracz odradza się po śmierci, ale planeta nie jeśli jej zdrowie spadnie do zera, koniec gry. 🌍💥

### ⚔️ Space Invader Normal
Klasyczny tryb na jedno życie. Pokonaj wszystkie fale wrogów, aby wygrać - jeśli zginiesz albo wróg przebije Twoją orbitę, **game over**. Prosto, szybko, bez litości. 💀

### ♾️ Space Invader Endless
Fala za falą, bez końca. Po każdej pokonanej fali poziom trudności rośnie. 📈🔥
